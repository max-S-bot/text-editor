# text-editor

the plan is to make a text editor that runs in the browser and 
makes get and post requests to a locally hosted web server or electron
using fetch

- first time using node in a real project
- first time using npm to manage JS dependencies (codemirror API)
- learned how to use esbuild so that I could actually use said frontend dependencies
- better understanding of HTTP
